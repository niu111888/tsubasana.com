# tsubasana.com

大原翼の個人サイト。GitHub Pages でホスティングしています（`main` にプッシュすると自動で公開反映）。

## しくみ

- `index.html` / `style.css` — サイト本体（1ページ構成、日英中の3言語対応）
- `content/ja.json` / `en.json` / `zh.json` — 各言語の文章。ここを書き換えるとサイトの文章が変わります
- `content/site.json` — メール・SNSリンク、ピアノ動画、旅の写真の設定
- `assets/` — 写真や動画などのアップロード先
- `.pages.yml` — CMS（Pages CMS）の管理画面設定

## 編集のしかた（CMS）

1. <https://app.pagescms.org> を開く
2. **Sign in with GitHub** でログイン（`niu111888` アカウント）
3. リポジトリ `niu111888/tsubasana.com` を選ぶ
4. 左メニューから編集したい項目を選んで保存
   - **サイト設定（動画・写真・連絡先）** — ピアノ動画のYouTube URL、旅の写真、メール・SNSリンク
   - **文章（日本語）／（英語）／（中文）** — 各言語の文章
5. 保存すると自動でコミットされ、1〜2分でサイトに反映されます

## よくある編集

- **ピアノ動画を載せる** — サイト設定 →「ピアノ動画」→ YouTubeのURLを貼るだけ
- **旅の写真を追加** — サイト設定 →「旅の写真」→ 写真をアップロードしてキャプションを入力
- **連絡先を設定** — サイト設定 → メール / Instagram / X のURLを入力（空欄のボタンは表示されません）
